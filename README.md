#Teslo API

1. Clonar proyecto
2. ``` npm i ```
3. Clonar archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables
5. Levantar base de datos con ```docker-compose up -d```
6. Iniciar modo desarrollo ```npm run start:dev```
